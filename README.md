# CricCircle 🏏

CricCircle is a professional cricket discussion platform where users can join as moderators or analysts to participate in structured, high-quality discussions.

## Features (MVP 1)
- Google Authentication (planned)
- Role-based access (Moderator / Analyst)
- Live discussion sessions
- Rating system
- Moderation controls

## Tech Stack

- Frontend: (to be added)
- Database: PostgreSQL (planned)
- DevOps: Docker, GitHub Actions (planned)
